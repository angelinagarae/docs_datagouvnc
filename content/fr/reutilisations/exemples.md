# Réutilisations

## Exemples de projets

Voici quelques exemples de projets qui utilisent les données disponibles sur **data.gouv.nc** :

- **OpenTransport NC** : une application mobile qui utilise les données de transport pour fournir des informations en temps réel sur les horaires de bus.
- **CalédoClimat** : un tableau de bord interactif qui présente les données climatiques de la Nouvelle-Calédonie.
  
Vous avez développé un projet avec nos données ? Soumettez-le via la section **Réutilisations** pour le partager avec la communauté.
