    Ajouter un jeu de données #

    Vous souhaitez partager un jeu de données public ? Voici les étapes pour contribuer :

    1. Rendez-vous dans la section Ajouter un jeu de données.
    2. Remplissez le formulaire avec les informations requises : titre, description, licence, et téléchargez vos fichiers.
    3. Cliquez sur Publier pour partager vos données avec la communauté.

    Assurez-vous que vos données sont bien structurées et correctement documentées avant de les publier.
